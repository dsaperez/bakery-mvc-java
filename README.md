# Bakery Management System 🚀

An enterprise management system for bakeries and pastry shops developed in *Java* utilizing robust software engineering practices and clean architecture.

## 🏗️ Architecture & Design

The system strictly adheres to the *MVC (Model-View-Controller)* pattern to decouple the business logic from the user interface. It incorporates advanced structural patterns including *DTOs (Data Transfer Objects)* and *DataMappers* to isolate data persistence components from the presentation layer.

## 🛠️ Core Features
* *Inventory Management:* Precise control over perishable goods with aging/deterioration tracking.
* *Payroll & Staff Administration:* Daily salary calculation system customized for distinct bakery roles.
* *Suppliers & Sales Control:* Automated management of industrial and artisan suppliers alongside transactions.
* *Robust Exception Handling:* Tailored validation suite (ExceptionChecker) preventing data type mismatches or negative inputs.
